# ModelWatch

ModelWatch is a native macOS menu bar application that helps users monitor AI tool usage, subscription costs, productivity metrics, and future API consumption analytics.

## Why ModelWatch?

AI users often switch between multiple tools such as:

* ChatGPT
* Claude
* Cursor
* Codex
* VS Code
* Gemini
* GitHub Copilot

Tracking activity, subscriptions, and future API spending across these tools can become difficult.

ModelWatch aims to provide a centralized dashboard that helps users understand how they interact with AI systems throughout the day.
